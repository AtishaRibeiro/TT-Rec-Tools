How To

Ghost Manager

Ghost Manager allows you to import and export ghost files from an mkwii savefile (rksys.dat).

How to find your rksys.dat file

Dolphin

Right click on your mkwii game in Dolphin and click on the option Open Wii Save Folder. This will bring you to a folder which contains the rksys.dat file.

Console

There are 2 main methods to get your savefile on console. Choose whichever one you prefer.

1. Using Riivolution you can enable the option save to sd card. This will create an rksys.dat file on your sd/usb next time you launch the game (if it hasn't been created yet.) The rksys.dat file will be located in riivolution\save\[GAME_ID] on your sd/usb. GAME_ID varies depending on the region your game is from.

2. Using SaveGame Manager GX you can export/import your savefile.

Using the tool

Once you have your savefile you can import it into the tool either by selecting the file after clicking Upload rksys.dat file or dragging and dropping the file on the upload region. You'll only be able to work with the licenses that are available in the savefile you uploaded. If for example your rksys.dat contains 2 licenses you'll only be able to import/export ghosts from those 2 licenses. Optionally you can use a blank savefile which has 4 licenses with no ghosts.

Importing ghosts

Press Upload RKG and select as many ghosts as you want. Alternatively, drag and drop the ghosts into the upload region. Select the license where you want your ghosts to be imported. You also have the option to choose whether the ghosts should be imported as pb's or downloaded ghosts. The following table shows the differences between these 2 types.

Personal Best Downloaded
Max 1 per track Max 32 per track
Can be overwritten by setting a pb Can't be overwritten by setting a pb
Visible on all 4 licenses Only visible on the current license
Visible on the MK channel Not visible on the MK channel

Ironically, downloaded ghosts are not visible in the MK channel, the reason for this kind of complicated.

Keep in mind that you get 32 download ghost slots for the entire license. If you use all 32 on 1 track you won't have any more left for other tracks!

Exporting/Deleting ghosts

You can select ghosts and then either delete or export the selected ghosts. Exporting will extract those ghosts from the savefile download them for you. Deleting a ghost deletes it from the savefile.

Downloading your new savefile

When you're ready importing/deleting ghosts, press Save & Download to download the edited savefile. Then replace your old rksys.dat with the one you just downloaded.
