How To

Ghost Manager

Ghost Manager allows you to import and export ghost files from an mkwii savefile (rksys.dat).

How to find your rksys.dat file

Dolphin

Right click on your mkwii game in Dolphin and click on the option Open Wii Save Folder. This will bring you to a folder which contains the rksys.dat file.

Console

There are 2 main methods to get your savefile on console. Choose whichever one you prefer.

1. Using Riivolution you can enable the option save to sd card. This will create an rksys.dat file on your sd/usb next time you launch the game (if it hasn't been created yet.) The rksys.dat file will be located in riivolution\save\[GAME_ID] on your sd/usb. GAME_ID varies depending on the region your game is from.

2. Using SaveGame Manager GX you can export/import your savefile.

Using the tool

Once you have your savefile you can import it into the tool either by selecting the file after clicking Upload rksys.dat file or dragging and dropping the file on the upload region. You'll only be able to work with the licenses that are available in the save file you uploaded. If for example your rksys.dat contains 2 licenses you'll only be able to import/export ghosts from those 2 licenses.

Importing ghosts

Simply press Upload RKG and select as many ghosts as you want. You can import up to 4 ghosts per track (1 per license.) In theory you could thus import 128 ghosts at once (32 per license.) Once you've imported your ghosts, download the file and replace your old rksys.dat with the one you just downloaded. Next time you launch the game your ghosts should be there!

Exporting ghosts

When exporting ghosts you'll be able to see all ghosts present in the save file. Simply select the ghost files you wish to export and press Export & Download. This will convert all ghost files to staff ghosts and put them in a zip file.
