How To

GCT Maker

A very simplistic tool that makes it easy to quickly create a GCT file.

Overview

Game ID

Simply fill in the ID you're making the GCT for. This tool works for any Wii game but the MKWii codes are given here:

  • RMCP01 for PAL
  • RMCE01 for NTSC-U
  • RMCJ01 for NTSC-J
  • RMCK01 for NTSC-K

Adding a code

Put the code contents in the right hand box and fill in any title you want (not necessary.) You can add more codes if you want. You can also insert all your codes at once by pasting them underneath each other.

Deleting/Editing

To delete a code simply press the X button.
Editing a code can be done by clicking on the name of the code in the left box, modifying the code and then pressing Edit Code.

Downloading the GCT file

When you're done press the Download GCT button.
