How To

MSG Editor

This tool generates a Gecko code that allows you to modify in game text.

Overview

What is a message?

MKWii uses messages to manage in game text. Every message has an ID and a text. This way the game can work with ID's only and fill in the right text depending on the language used. For example, Press the A Button and its french counterpart Appuyez sur le bouton A both have ID 07E1.

Finding the right message

If you know the ID of the message you want to change you can simply fill in the MSG ID field. If it's a valid ID the corresponding text will appear which you can then edit. If you do not know the ID you can simply write the text you're looking for in the Text field. A drop-down will appear with some suggestions. By clicking on a suggestion the ID will be automatically filled in.

Language

The language of the text field can be changed. This does not have an effect on the generated code but can help with searching for the right text/ID.

Generating the code

You can control the amount of messages you want to modify by pressing the + and - buttons. Once done simply press Start to generate the code and Copy to copy the code to your clipboard.

Cheat sheet

You may notice that some messages contain weird character sequences. These are called Escape Sequences and they allow you to modify the text in different ways. For example, looking at ID 1777 we see the corresponding text is H{001A080200110000} Worldwide Top 10. Removing the H{} and everything in between will remove the track name in front of Worldwide Top 10. This tells us that the game replaces that particular sequence with the relevant track.

Colours & Font size

Both the colour and font size modifier work the same way: They modify all text that follows. ${RED3}Mario${GREEN}Luigi will thus make the game show MarioLuigi.

Unicode

Only a limited amount of Unicode characters are supported. Here is a list of characters the game uses. It's possible however that the game supports more characters than shown there.

Hex

You can insert literal hex strings using the H{} syntax. I would only recommend using this if you understand how escape sequences work and know what you're doing. For more info on escape sequences check this page.
