How To

Record with Dolphin

This tutorial will show you how to record a run using Dolphin, the Wii/GCN emulator.

Necessities

There are 2 things that are quintessential if you want to record on Dolphin: Dolphin (obviously) and an MKWii ISO.

Downloading Dolphin

Dolphin can be downloaded at dolphin-emu.org. If you don’t know what version to download, choose Dolphin 5.0. If you want to record in very high quality, download version 4.0-8572 (Warning: this can create very large video files.) T

Getting your ISO

The recommended way of obtaining an ISO is to create a backup of a real MKWii disc. This can be achieved with the Homebrew app CleanRip. There are a few tutorials online on how to use CleanRip such as this one. Note that, unlike what is suggested in the video, you can actually use sd’s/usb’s smaller than 8GB by setting the chunk size to something that will fit on your sd/usb.

Running your ISO

In order to run the ISO in Dolphin, place the ISO in a folder of your choosing and then point Dolphin to that folder. This can be done in Dolphin by going to Options > Configuration > Paths > Add… and selecting your folder. The game should now appear in Dolphin and can be launched by selecting it and pressing Play.

Ghost Files

Getting your ghost file

All ghosts are saved as .rkg files so you’ll need this file you’re trying to record. If the ghost was set on CTGP-R it is as simple as going to Chadsoft, finding your time and clicking the download link at the bottom.

Importing your ghost file

To get your rkg file into the game on Dolphin we’ll need to import it into the savedata of the game. To find this file simply right click on the game in Dolphin and select Open Wii Save Folder. There you will find an rksys.dat file. From there, use the Ghost Manager on this site as explained in the how-to section .

Recording

Finally, we’ll get to recording. The best way to record Dolphin footage is to dump all audio and frames. This way it doesn’t matter how slow Dolphin runs so don’t worry if your pc is not the best.

Graphics settings

You can access the graphics settings by going to Options > Graphics Settings. The recommended graphics settings are shown below. The Internal Resolution is set to 4x native but you can set this to your desired value.

Dumping frames/audio

When you’re ready to start recording, pause the game. This will ensure that the audio and frames sync up. Enable audio and frame dumping clicking Movie > Dump Audio and Movie > Dump Frames. Resume the game by pressing Play. The game will run notably slower because Dolphin is working hard to dump everything. When you want to stop the recording, disable the dumping the same way you enabled it.

To find your dumped audio and frames, go to your Dolphin folder (which is usually located in your Documents folder) where you will find a Dump folder. Your dumped frames will be in Frames and the audio will be in Audio. In the Audio folder you’ll find 2 audio files: dspdump and dtkdump. You’ll only need dspdump. Copy the frames and audio to somewhere else as next time you start a dump, they will be overwritten.

Note that if you’re using Dolphin 4.0-8572 the frames will be split over multiple files which can be combined in a video editor.

Editing the video

Now that you have your frames and audio, use a video editor of your choice and import them. If the game was paused when you started the frame and audio dump the should sync up automatically.
