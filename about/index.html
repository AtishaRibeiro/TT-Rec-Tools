Info

This site was created by WhatisLoaf with the intent to bring together several tools related to recording runs for Mario Kart Wii. It was not made with mobile devices in mind so it's best used on a pc.

All scripts on this site run client-side which means no files uploaded are actually sent to the server. The entire source code is also available on github if you wish to check it out.
